平板电子相册

v1.0.0 · 单文件 HTML · 完全本地运行

把闲置的手机、平板、显示器改造成一台优雅的电子相框。零依赖、零上传、零追踪—— 所有图片都通过浏览器原生文件系统 API 在本地读取，绝不上传到任何服务器。

使用提示
· 首次打开请使用 Chrome / Edge / Opera 浏览器（基于 Chromium 内核），需要 HTTPS 或 localhost 才能使用文件夹 API。
· 推荐把 HTML 文件放在 https:// 站点或本地双击（Chrome 在 file:// 也支持，但部分 API 受限）。
· 添加文件夹后会自动递归扫描子目录，所有图片会自动去重。

平板电子相册

把你的平板、手机、显示器改造成一个会呼吸的电子相框。
支持嵌套文件夹、多文件夹混播、横竖屏切换、屏保时钟。

建议使用 Chrome / Edge 浏览器
键盘快捷键：Space 播放 · ← → 切换 · F 全屏 · H 隐藏 UI
⚠ 当前浏览器可能不完全支持文件夹选择 API，建议使用 Chrome / Edge 最新版。
⚠ 检测到你在 内置浏览器 中打开
微信/QQ/微博等内置浏览器 不支持直接选整个文件夹，只能逐张选图（最多 100 张）。
想要 一键选整个相册，请点击右上角 ··· → 在浏览器中打开，或复制链接到 Safari / Chrome 打开。
